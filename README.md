# 📚 C++ Library Management System

A console-based Library Management System built using **C++ and Object-Oriented Programming (OOP)**.

This project demonstrates real-world library operations using a menu-driven system.

---

## ✨ Features
- Add new books
- Display all books
- Search books by ID or name
- Issue books
- Return books
- Delete books
- File handling support (optional)

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Arrays
- File Handling (optional)
- Console Application

---

## 📌 Purpose of Project
This project was built to practice:
- OOP concepts in C++
- CRUD operations
- Menu-driven programming
- Problem solving
